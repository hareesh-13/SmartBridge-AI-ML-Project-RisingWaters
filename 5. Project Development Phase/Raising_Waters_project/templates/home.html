About The Project

What is Rising Waters?

Rising Waters is a state-of-the-art predictive platform that utilizes historical meteorological and hydrological parameters to compute the probability of regional flooding. By deploying robust machine learning algorithms, the system aids in forecast models that translate complex data matrices into actionable insights.

Importance of Flood Prediction

Proactive warnings give responders the buffer time needed to construct barriers and save agricultural fields, commercial hubs, and residences.

How ML Solves It

Our underlying Decision Tree Classifier examines sub-divisional rainfall distributions to recognize high-risk trends that standard heuristic logic might overlook.
