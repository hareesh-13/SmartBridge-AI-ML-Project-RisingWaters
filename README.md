# Project Overview

**Project Title:**
**Rising Waters вЂ“ Intelligent Flood Prediction and Early Warning System**

## Objective

The objective of this project is to predict flood occurrences using Machine Learning based on historical weather and rainfall data. The system provides early warnings that help improve disaster preparedness and decision-making.

---

# Features

- Flood Prediction using Machine Learning
- Decision Tree Classifier
- User-friendly Flask Web Application
- Responsive UI
- Input Validation
- Fast Prediction Results
- Historical Dataset Analysis
